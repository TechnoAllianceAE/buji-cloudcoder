Usage Guide

This guide covers everything from basic commands to advanced features.

Basic Commands

Start using free-code with these essential commands:

Interactive Mode

./cli-dev

Starts an interactive chat session where you can have conversations with the AI.

One-shot Mode

./cli-dev -p "what files are in this directory?"

Ask a single question and get an immediate response.

Specify a Model

./cli-dev --model claude-opus-4-6

OAuth Login

./cli-dev /login

Authenticate with your model provider using OAuth.

For Non-CLI Users

If you're new to command-line tools, here's what you need to know:

  • Everything you type after './cli' is a command or option
  • The '-p' flag lets you ask a question directly
  • Running './cli' alone starts interactive mode where you can chat
  • Press Ctrl+C to exit at any time

Environment Variables

Configure free-code using environment variables:

Variable Purpose
ANTHROPIC_API_KEY Anthropic API key
CLAUDE_CODE_USE_OPENAI=1 Use OpenAI Codex provider
CLAUDE_CODE_USE_BEDROCK=1 Use AWS Bedrock provider
CLAUDE_CODE_USE_VERTEX=1 Use Google Vertex AI provider
CLAUDE_CODE_USE_FOUNDRY=1 Use Anthropic Foundry provider
