Usage Guide

Modes

Interactive REPL

./bc2

One-Shot

./bc2 -p "explain cmd/bc2/main.go"

Piped Input

echo "list Go files" | ./bc2

JSON Output

./bc2 -p "summarize this repo" -json

Structured Output

./bc2 -p "list functions" -schema '{"type":"object","properties":{"fns":{"type":"array"}}}'

Project Instructions

Create BC2.md in your project root. It is loaded into the system prompt automatically.
