BujiCloudCoder (bc2)

Open-source AI coding CLI built in Go. Single binary, zero dependencies, 32 tools.

Get Started
⚡

Lightning Fast

Single 12MB Go binary. Under 50ms startup. 10x less RAM than Node.js alternatives.

🔧

32 Built-in Tools

Bash, file ops, web search, agents, tasks, MCP, LSP, cron, team coordination, and more.

🤖

Real Sub-Agents

Spawn autonomous sub-agents with their own conversation context, tools, and token budgets.

🔌

MCP Support

Connect to MCP servers via stdio, HTTP, or SSE. Discover tools and resources dynamically.

🔒

Security First

Permission system with dangerous command detection, path protection, and authenticated bridge API.

🌐

Multi-Provider

Works with Anthropic, AWS Bedrock, Google Vertex, Azure Foundry, and OpenAI-compatible APIs.

Quick Start

git clone https://github.com/TechnoAllianceAE/buji-cloudcoder.git cd buji-cloudcoder go build -o bc2 ./cmd/bc2/ export ANTHROPIC_API_KEY=sk-ant-... ./bc2
