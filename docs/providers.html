API Providers

bc2 supports multiple LLM API providers. Set the appropriate environment variables to switch.

Anthropic (Default)

export ANTHROPIC_API_KEY=sk-ant-api03-...
VariableDescription
ANTHROPIC_API_KEYAPI key
ANTHROPIC_BASE_URLCustom endpoint (optional)
ANTHROPIC_DEFAULT_MODELOverride default model

AWS Bedrock

export CLAUDE_CODE_USE_BEDROCK=true export AWS_ACCESS_KEY_ID=... export AWS_SECRET_ACCESS_KEY=... export AWS_REGION=us-east-1

Google Vertex AI

export CLAUDE_CODE_USE_VERTEX=true export ANTHROPIC_VERTEX_PROJECT_ID=my-project export CLOUD_ML_REGION=us-east5

Azure Foundry

export CLAUDE_CODE_USE_FOUNDRY=true export ANTHROPIC_FOUNDRY_API_KEY=... export ANTHROPIC_FOUNDRY_RESOURCE=my-resource

OpenAI-Compatible

export CLAUDE_CODE_USE_OPENAI=true export OPENAI_API_KEY=... export OPENAI_BASE_URL=https://api.openai.com/v1

Works with any OpenAI-compatible API (OpenAI, Ollama, LM Studio, vLLM, etc.)

Provider Detection

bc2 checks environment variables in this order: Bedrock → Vertex → Foundry → OpenAI → Anthropic (default). The first matching provider is used.
