Model Providers

free-code supports five different API providers. Choose the one that fits your needs.

Anthropic (Direct API)

Use Anthropic's first-party API directly. This is the default provider.

Model ID
Claude Opus 4.6 claude-opus-4-6
Claude Sonnet 4.6 claude-sonnet-4-6
Claude Haiku 4.5 claude-haiku-4-5
export ANTHROPIC_API_KEY="your-api-key"

OpenAI Codex

Use OpenAI's Codex models for code generation. Requires a Codex subscription.

Model ID
GPT-5.3 Codex (recommended) gpt-5.3-codex
GPT-5.4 gpt-5.4
GPT-5.4 Mini gpt-5.4-mini
export CLAUDE_CODE_USE_OPENAI=1

AWS Bedrock

Route requests through your AWS account via Amazon Bedrock.

export CLAUDE_CODE_USE_BEDROCK=1
export AWS_REGION="us-east-1"

Uses your standard AWS credentials (environment variables, ~/.aws/config, or IAM role).

Google Cloud Vertex AI

Route requests through your GCP project via Vertex AI.

export CLAUDE_CODE_USE_VERTEX=1

Uses Google Cloud Application Default Credentials (gcloud auth application-default login).

Anthropic Foundry

Use Anthropic Foundry for dedicated deployments.

export CLAUDE_CODE_USE_FOUNDRY=1
export ANTHROPIC_FOUNDRY_API_KEY="..."
