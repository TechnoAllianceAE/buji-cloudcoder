Installation Guide

Getting free-code up and running is simple. Follow these steps:

Install Bun

free-code requires Bun runtime version 1.3.11 or higher.

curl -fsSL https://bun.sh/install | bash

Clone the Repository

git clone https://github.com/paoloanzn/free-code.git
cd free-code

Build

bun install
bun run build:dev:full

This builds with all 54 experimental features enabled.

Run

./cli-dev

Note: Windows users should use WSL (Windows Subsystem for Linux).

Quick Install (Automated)

Use our one-liner installer that handles everything automatically:

curl -fsSL https://raw.githubusercontent.com/paoloanzn/free-code/main/install.sh | bash

This script checks your system, installs Bun if needed, clones the repo, builds with all experimental features enabled, and symlinks free-code on your PATH.

Post-Installation

After installation, you need to authenticate with your preferred model provider:

./cli-dev /login

Or set your API key as an environment variable:

export ANTHROPIC_API_KEY="your-api-key-here"
