Installation

Prerequisites

  • Go 1.23+ — go.dev/dl
  • API Key — From your LLM provider
  • Git

Build from Source

git clone https://github.com/TechnoAllianceAE/buji-cloudcoder.git cd buji-cloudcoder go build -o bc2 ./cmd/bc2/

Cross-Compile

# Linux GOOS=linux GOARCH=amd64 go build -o bc2-linux ./cmd/bc2/ # macOS Apple Silicon GOOS=darwin GOARCH=arm64 go build -o bc2-mac ./cmd/bc2/ # Windows GOOS=windows GOARCH=amd64 go build -o bc2.exe ./cmd/bc2/ # ARM (Raspberry Pi) GOOS=linux GOARCH=arm64 go build -o bc2-arm ./cmd/bc2/

API Key Setup

export ANTHROPIC_API_KEY=sk-ant-api03-...

Or use /login for OAuth, or add to ~/.bc2/settings.json.

Verify

./bc2 -version ./bc2 -doctor

Optional: Ripgrep

bc2 auto-detects rg for 10-100x faster grep. Install from github.com/BurntSushi/ripgrep.
